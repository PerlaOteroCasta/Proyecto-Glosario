Glosario de Fronted

Comando
Instrucción que un usuario ejecuta en una terminal o consola para realizar tareas específicas como iniciar un servidor o usar Git.
Commit
Punto de guardado en un sistema de control de versiones como Git, que almacena cambios realizados en un repositorio.
CSS
(Cascading Style Sheets) Lenguaje que define el estilo y diseño de un documento HTML, como colores, fuentes y disposición.
Discord
Plataforma de comunicación en tiempo real, útil para colaborar en proyectos web y discutir sobre desarrollo.
Etiquetas (Tags)
Elementos de HTML que definen la estructura de un documento.
Etiquetas Semánticas
Estiquetas HTML que describen claramente su propósito, mejorando la accesibilidad y SEO.
Extensiones
Complementos que amplían las funcionalidades de herramientas como editores de código (por ejemplo, extensiones de VS Code para HTML o CSS).
Float
Propiedad CSS utilizada para posicionar elementos a la izquierda o derecha dentro de un contenedor.
Formulario
Elemento HTML (form) utilizado para recopilar datos del usuario mediante campos como input o textarea.
Fusión (Merge)
Acción en Git que combina cambios de una rama con otra, uniendo sus contenidos.
Git
Sistema de control de versiones que rastrea los cambios en archivos y facilita la colaboración entre desarrolladores.
GitHub
Plataforma que aloja repositorios Git y facilita la colaboración y gestión de proyectos.
Herencia
En CSS, propiedad mediante la cual los estilos de un elemento padre se transmiten a sus elementos hijos.
HTML
(HyperText Markup Language) Lenguaje de marcado utilizado para estructurar el contenido en la web.
IDE
(Integrated Development Environment) Entorno de desarrollo que integra herramientas como editor de código, depurador y terminal (por ejemplo, Visual Studio Code).
Inspector Web
Herramienta del navegador para analizar y modificar el HTML, CSS y JavaScript de una página web en tiempo real.
Inteligencia Artificial (IA)
ETecnología que permite a las máquinas realizar tareas que requieren inteligencia, como el análisis automatizado del comportamiento de usuarios en sitios web.
JavaScript
Lenguaje de programación utilizado para añadir interactividad y funcionalidad a páginas web.
LLMs
(Large Language Models) Modelos de IA avanzados que procesan texto para tareas como generación de código o asistencia en desarrollo.
Main
Etiqueta semántica de HTML que define la sección principal del contenido de una página.
Modelo de Caja
Concepto de CSS que describe cómo se calculan los tamaños y márgenes de los elementos HTML.
Node.js
Entorno de ejecución para JavaScript del lado del servidor, usado para construir aplicaciones web escalables.
Position
Propiedad CSS que define cómo se posiciona un elementos en la página
Prompt
Instrucción o texto de entrada utilizado para interactuar con sistemas de IA o scripts automatizados.
Ramas (Branches)
Copias paralelas de un repositorio Git que permiten trabajar en características o cambios sin afectar el código principal.
README.md
Archivo de texto en formato Markdown que proporciona información sobre un proyecto, como su propósito y cómo usarlo.
Repositorio
Contenedor de archivos y versiones de un proyecto, gestionado por sistemas como Git.
Sprint
Periodo de tiempo corto y definido, durante el que un equipo completa una cantidad específica de trabajo.
Staging Area
Área temporal en Git donde se preparan los cambios antes de realizar un commit.
Visual Studio Code
Editor de código altamente configurable utilizado para desarrollar en HTML, CSS, JavaScript y más.
